## 🔧 COMO USAR O ARQUIVO

### Localização:
```
lib/material-estudo-completo.ts
```

### Estrutura de cada tópico:

```typescript
{
  id: "pt-sup-coesao",           // ID único
  title: "Coesão e Coerência",   // Título
  subject: "Português",           // Disciplina
  level: "superior",              // Nível (superior/medio)
  
  whatYouNeedToKnow: string,       // 1 parágrafo
  whyItMatters: string,            // Por que aparece em prova
  stepByStep: string[],            // 5 passos estruturados
  visualGuide: string,             // Fluxogramas/tabelas ASCII
  comparativeTable: {...},         // Tabela de comparação
  provenTricks: string[],          // 6 "macetes" de prova
  questionsResolved: Question[],   // 4 questões com estratégia
  cheatSheet: string              // 1 página de memorização
}
```

### Importar em componentes React:

```typescript
import { allStudyMaterials } from '@/lib/material-estudo-completo';

// Filtrar por disciplina
const portuguesMaterials = allStudyMaterials.filter(
  m => m.subject === 'Português'
);

// Filtrar por nível
const superior = allStudyMaterials.filter(
  m => m.level === 'superior'
);

// Buscar específico
const coesao = allStudyMaterials.find(
  m => m.id === 'pt-sup-coesao'
);
```
